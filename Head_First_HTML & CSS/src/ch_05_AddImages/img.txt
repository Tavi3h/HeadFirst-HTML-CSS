关于<img>标签：

<img>元素是一个内联元素，它不会再前面或后面插入换行。
<img>元素是一个void元素
src = "pathname/filename" 
alt = "..." 当图片不能显示时，显示alt中的文本。

可选属性：
pathname可以是URL，也可以使相对链接
width = "..."  定义宽度 单位：像素
height = "..." 定义高度 单位：像素
即使指定了宽度和高度，浏览器也会下载完整的图像然后再调整大小

常用图像格式区别：
---jpeg：
	最适合连续色调图像，如照片。
	可以表示包含多大1600万种不同颜色的图像。(0,0,0) --> (255,255,255)
	这是一种有损格式，因为缩小文件大小时会丢失图像的一部分信息。
	不支持透明度。
	文件比较小，以便Web页面更高效的显示。
	不支持动画。
---png：
	最适合单色图像和线条构成的图像。
	包含上百万种不同颜色的图像。png有三种 png-8、png-24、png-32，取决于你需要表示多少中颜色。
	png会压缩文件来缩小大小，不过不会丢失信息。所以是一种无损格式。
	允许将颜色设置为“透明”，使图像下面的东西可以显示出来。
	与jpeg相比，大小更大一些，不过这取决于使用颜色数目。可能比相应的gif小也可能大。
---gif：
	类似于png，gif最适合单色图像和线条构成的图像。
	gif表示最多256种不同颜色的图像。
	gif也是一种无损格式。
	gif也支持透明度，不过只允许一种颜色设置为“透明”。
	gif往往比相应的jpeg大。
	支持动画。
