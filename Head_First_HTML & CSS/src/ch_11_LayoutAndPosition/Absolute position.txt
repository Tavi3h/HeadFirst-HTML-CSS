使用绝对定位设置元素的位置。
#sidebar {
	position: absolute; /*首先使用position属性指定这个元素要绝对定位*/
	/*设置top和right属性，也可以是bottom、left*/
	top: 100px;
	right: 200px;
	/*为<div>指定一个宽度*/
	width: 280px;

一个元素绝对定位时，浏览器首先要做的是将它从流中完全删除，然后浏览器将这个元素放置在指定位置。

由于边栏在流之外，其它元素甚至不知道有这样一个元素，所以其它元素会将边栏完全忽略。
流中的元素不会将其内敛内容围绕在一个绝对定位的元素周围。因为它们不知道这儿有个绝对定位的元素。

可以对任何元素指定绝对位置，包括块元素和内联元素。

position属性的4个值：
	static：position的默认值，元素会放在正常的文档流中，要有浏览器决定静态定位元素的位置。
	absolute：绝对定位。
	fixed：固定定位。元素放在相对于浏览器窗口（而不是页面）的一个位置上。
			采用固定定位放置内容，它会一直留在你原先指定的位置，不再移动，即使你滚动页面它也原地不动。
	relative：相对定位会让元素正常地流入页面，不过在页面上显示前要进行偏移。

绝对位置的上下关系可以用z-index指定，值大的在上。
只有使用绝对定位的元素才有z-index属性。
